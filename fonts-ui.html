This assignment focus on fonts. We've learned about the different styles of fonts and how to use them properly.
We also learn about UI cards design, and how important is font hierarchy so the web content is easy to understand.
This assignment helped me to be more consistent with the use of fonts, to use the same font with different styles rather than different fonts, and to style line-height and kerning depending on each block of text, instead of using the CSS presets.
We saw how small changes on font usage can improve the user experience on our website.
