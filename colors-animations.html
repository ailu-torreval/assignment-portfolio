COLOR PALETTE

In this assignment, we create a color palette to use on a dummy website. I usually only ver neutral color palettes with three or four colors only, so for this assignment, I tried to work outside my safe zone, and implement six colors. I used the accessibilities tool from Adobe we learned in class, to check the contrast of the colors was enough. We also implement a Dark Theme that also helped me to have a better understanding of colors usage. We learned the basics of WAAPI animations, but I couldn't implement them on this web.
I'm happy with the outcome, but I wouldn't implement that many colors in the future.
